# iRacingAnalyser

A desktop tool that converts iRacing IBT telemetry files into CSV format and visualises telemetry data.

## Features
- Session analysis (Practice | Qualifying | Race)
- Telemetry visualisation (graphs)
- Drag & drop UI
- File picker support

## Tech Stack
- Python
- PySide6
- pandas
- pyirsdk
- matplotlib

python -m venv venv
venv\Scripts\activate
